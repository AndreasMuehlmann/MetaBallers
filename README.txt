METAbALLERS
A Project in wich we make a meta ball simulation.
How it could look: "https://github.com/AndreasMuehlmann/MetaBallers/files/8873628/Metaballs.Draw.1.pdf".


QUICKSTART
Clone the repository with "git clone https://github.com/AndreasMuehlmann/MetaBallers.git"

To only run:
If your on Windows, run ".\bin\MetaBallers.exe".
If your on Linux, run ".\bin\MetaBallers".

To Recompile and run (required is g++, make and SDL2):
If your on Windows, run "make --file=MakefileWindows".
If your on Linux, run "make --file=MakefileLinux".
